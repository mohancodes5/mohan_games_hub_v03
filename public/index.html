Game Hub

Solo or play with friends using a room code

Play anywhere (recommended)

  1. Deploy this app once to the internet (free): open DEPLOY.md in the project folder and follow Render steps.
  2. You get a link like https://your-game.onrender.com — open it on any phone (Wi‑Fi or mobile data).
  3. Bookmark it or use Add to Home Screen so it feels like an app.
  4. With friends: share that same link. One taps Create room and sends the 6‑letter code; the other taps Join room.

At home only (no deploy)

  1. PC: npm install then npm start.
  2. Phone on same Wi‑Fi: http://YOUR_PC_IP:3000 (see ipconfig for IPv4).

Free cloud hosting may “sleep” when idle; the first open after a while can take ~30–60s, then it’s normal.
